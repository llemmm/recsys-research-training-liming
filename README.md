## 使用方式

1. 在 Obsidian 中选择“打开本地仓库”，打开本目录。
2. 每天复制 `daily_logs/TEMPLATE.md`，并重命名为当天日期。
3. 将课程作业放入 `assignment/`，长期笔记放入 `knowledge_base/`。
4. 每次学习后提交 Git 记录；确认内容无误后同步到 GitHub。

## 记录原则

- 记录可复现的信息：环境、命令、配置、日志和结果。
- 先写事实，再写判断；对 AI 生成内容进行人工核对。
- 问题要包含现象、已经尝试的方法、当前猜测和下一步。
- 每周回顾一次，合并重复笔记并更新学习计划。
