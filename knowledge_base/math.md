# 数学基础笔记

## 学习范围

### 线性代数

- 向量、矩阵、张量及其形状。
- 点积、矩阵乘法、转置和范数。
- Embedding 可以理解为把离散 ID 映射为可学习的稠密向量。
- 相似度常用点积或余弦相似度表示。

余弦相似度：

$$
\operatorname{cos}(\mathbf{x},\mathbf{y})=
\frac{\mathbf{x}^{\mathsf T}\mathbf{y}}
{\|\mathbf{x}\|_2\|\mathbf{y}\|_2}
$$

### 概率统计

- 条件概率、独立性和贝叶斯公式。
- 期望、方差、均值与置信区间。
- 训练集、验证集和测试集的作用。
- 随机种子只能提高可复现性，不能保证所有硬件与算子完全确定。

### 优化基础

- 损失函数衡量预测与目标之间的差异。
- 梯度描述损失对参数变化的局部敏感程度。
- 学习率决定每一步参数更新的尺度。
- 训练损失下降不等于泛化性能一定提高，需要观察验证集指标。

## 与推荐系统的联系

- 二分类任务常用二元交叉熵。
- 排序学习关注正负物品之间的相对顺序。
- Top-K 评价需要理解命中、相关物品数量和排名位置。
- 离线指标差异应结合数据划分、负采样和评价协议解释。
