# 个人知识库索引

本目录用于沉淀可以长期复用的概念、方法、命令和模板。课程日志记录“当天发生了什么”，知识库记录“以后仍然有用的内容”。

## 分类

- [Python 与 PyTorch](python.md)：Python 数据处理、张量、自动求导和训练流程。
- [数学基础](math.md)：线性代数、概率统计、优化和评价指标所需数学。
- [推荐系统](recsys.md)：任务、数据、模型、指标和实验范式。
- [论文阅读](paper_reading.md)：检索记录、文献卡片和精读模板。
- [科研工具](tools.md)：Obsidian、Git、GitHub、VS Code、PowerShell 和 Zotero。

## 维护规则

1. 每条笔记尽量回答一个明确问题。
2. 命令和代码要标注运行环境；实验结果要记录配置和随机种子。
3. 尚未确认的内容标记为“待验证”，避免把猜测写成结论。
4. 每周整理一次当天日志，将可复用内容迁移到相应知识库文件。
5. 重要资料记录来源、访问日期和链接，便于回查。
